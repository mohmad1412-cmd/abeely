# خطة تبسيط الكود

## الأهداف
- تقليل عدد أسطر الكود بنسبة 40-50%
- الحفاظ على جميع المميزات والخصائص
- تحسين قابلية الصيانة والقراءة

## التغييرات المخطط لها

### 1. Custom Hooks (تم ✅)
- `useAppState` - إدارة حالة التطبيق
- `useScrollPersistence` - حفظ موضع التمرير
- `useFilteredList` - فلترة وقائمة العناصر

### 2. Context API (تم ✅)
- `AppContext` - البيانات المشتركة بين المكونات

### 3. مكونات قابلة لإعادة الاستخدام (تم ✅)
- `FilterableList` - مكون موحد للفلترة والبحث

### 4. دمج المكونات المتشابهة (قيد العمل)
- دمج `MyRequests` و `MyOffers` في مكون واحد
- استخدام `FilterableList` في كلا المكونين

### 5. تبسيط App.tsx (قيد العمل)
- نقل المنطق إلى hooks مخصصة
- استخدام Context API بدلاً من تمرير props كثيرة
- دمج useEffect المتشابهة

## النتائج المتوقعة
- App.tsx: من 3630 سطر → ~1500-2000 سطر
- MyRequests.tsx: من 812 سطر → ~400 سطر (باستخدام FilterableList)
- MyOffers.tsx: من 715 سطر → ~400 سطر (باستخدام FilterableList)
- إجمالي التوفير: ~1500-2000 سطر
